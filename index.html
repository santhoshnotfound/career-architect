How It Works
1. Skill Detection
Paste your resume OR scan your GitHub profile. Both methods extract canonical skill names that match nodes in the knowledge graph.
2. Graph Gap Analysis
Your skills are compared against curated requirements for your chosen role. Missing skills are identified via set difference.
3. Shortest-Path Roadmap
NetworkX runs BFS on the prerequisite DAG to find the minimum learning path from what you know to each missing skill.
